#### 直感的に media query を書けるとは

[ついに Safari も。media query の範囲指定をより直感的に書ける記法が全ブラウザ対応へ](https://zenn.dev/moneyforward/articles/css-range-syntax)

- 以上や以下に加え、未満も定義できる
- コンテナクエリも考慮できる

```css
/* これまでは */
@media (min-width: 600px) and (max-width: 799px) {
}

/* 今後は */
@media (600px <= width < 800px) {
}
```
